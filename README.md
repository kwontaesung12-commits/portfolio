#   데이터 분석가 권태성 포트폴리오

## 소개
안녕하세요. 데이터로 세상을 이해하고 가치를 창출하는 **데이터 분석가 권태성**입니다.  
조직의 목표와 개인의 성장을 연결해 **데이터 기반 시너지**를 만드는 방법을 탐구하고 있습니다.  

**경영학 전공**과 **인공지능사관학교 데이터분석 1200시간 과정 수료**를 통해  
비즈니스 감각과 데이터 기술을 함께 갖추었습니다.  

현재는 데이터 분석뿐만 아니라 **백엔드, 클라우드, 임베디드 시스템, GPU 연산 최적화** 등  
데이터 생태계 전반의 기술을 확장하고 있습니다.
